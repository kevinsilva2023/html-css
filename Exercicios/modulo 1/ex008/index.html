Principais Formatações

Negrito/ Destaque

Nesta Frase temos um termo em negrito usando a tag "b" (não semântica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica)

Itálico/ Ênfase

Nesta Frase, temos um termo em italico usando a tag I (não semântica)

Nesta frase, temos um termo em ênfase usando a tag EM (semântica)

Marca Texto

Nesta frase, temos um termo em marcação usando a tag MARK

E no outro parágrafo temos outro texto marcado

Agora aepnas um outro teste para saber se está funcionando perfeitamente

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto Deletado

Podemos Marcar um texto como excluído para indicar que ele deve ser lido, mas não conseiderado

Para incluir a marcação de deletado é usado a tag DEL

Texto Inclído

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Para incluir a marcação de deletado é usado a tag INS

Existe também o sublinhado com a tag U (não semântica)

Texto Sobrescrito

Para inserir codigos ou até mesmo numeros "superiores", por exemplo:

X20 (X elevado à 20)

Para isso vamos usar a tag SUP

Texto Subscrito

Para inserir codigos ou até mesmo numeros "inferiores", por exemplo:

H20 (H20)

Para isso vamos usar a tag SUB

Semântica: tem siginificado
Não Semântica: não tem significado (não tem significado? É forma! é forma? é CSS)
