ANOTAÇÕES

Estilos InLine

Entrar dentro da TAG e codar a css

Ex:
h2 style="color: darkgray;" Capítulo 1.1

Estilos Interno

Consistes em alterar todas as tags da página de uma vez, de modo geral

Ex:

    style
        body {
            background-color: rgb(143, 147, 201);
            font-family: 'Times New Roman', Times, serif;
            font-size: 20px;
            }
        h1 {
            color:rgb(255, 255, 255);
            font-family: 'Times New Roman', Times, serif;
            background-color: midnightblue;
        }

Estilos Externos

Nada mais é que ultiluzar o mesmo layout do estilo internor, porém em uma aba propria "nome.css"
E vincular um link:css em casa aba nome.html

IMPORTANTE:
Sempre que for começar o documento da css utlizar a tag inicial 

                @charset "UTF-8";

E não se esquecer ultilizar o ;;;;;;;;; ao final do style.

CONSELHOS

Evite usar o InLine, à não ser que seja muuuuito necessario.
Usar os codigos Internos apenas para pequenas alterações especificas.
Use o Externo em todo custo, principalmente quando o site tiver mais de uma página.
