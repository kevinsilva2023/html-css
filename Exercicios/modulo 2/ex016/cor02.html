Harmônia das Cores

Cores Primarias

As cores primarias formam uma harmonia mais grosseira, mão casam muito bem, porém são importantes para representar as outras cores

Cores Secudárias

Tem quase o mesmo conseito das primarias, são muito dificeis de trabalhar

Cores Terciárias

Que são mistruras das primaria com as secundária, pode se dizer que são os tons pasteis

O recomendado são 4
